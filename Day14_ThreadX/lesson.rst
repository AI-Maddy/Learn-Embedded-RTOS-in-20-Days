Day 14 Lesson - ThreadX (Azure RTOS)
======================================

ThreadX (now Azure RTOS ThreadX) is Microsoft's industrial-grade RTOS with extensive certifications.

Key Features:
- Safety-certified (IEC 61508, DO-178B, etc.)
- Deterministic performance
- Priority inheritance
- Thread time-slicing
- Azure cloud integration

API Style:
- tx_thread_create()
- tx_semaphore_get()
- tx_queue_send()
- tx_mutex_put()

When to use: Safety-critical applications, medical devices, Azure IoT integration.
