Day 07 - Interrupt Handling with RTOS
======================================

Focus
-----
This day concentrates on ISR-safe design and deferred processing.

Learning Objectives
-------------------
- Understand the core concept and where it appears in embedded products.
- Build or reason about one practical implementation pattern.
- Identify failure modes and prevention checks for production firmware.

Deliverables
------------
- Concept notes in ``lesson.rst``.
- Practice problems in ``exercises.rst``.
- Guided answer strategy in ``solutions.rst``.

Definition of Done
------------------
- You can explain the concept without notes.
- You can map the concept to at least one real product scenario.
- You can list at least three verification checks for correctness.
