Day 10 Lesson - Build Systems, Ports, and Board Bringup
==========================================================

Integrating RTOS into new projects requires understanding build systems, porting procedures, and board-specific configuration.

Key Topics:
- CMake/Make build configuration
- RTOS port architecture
- BSP (Board Support Package) structure
- Clock configuration
- Startup code and linker scripts
- Debugging newly ported systems
