Day 13 Lesson - ChibiOS
========================

ChibiOS is a complete development environment with RTOS, HAL, and advanced features. Known for performance and clean architecture.

Key Features:
- Sub-microsecond context switch
- Static/dynamic objects
- Advanced synchronization (events, conditions)
- Comprehensive HAL
- File system support (FatFS integration)

Threading Model:
- Static threads (compile-time allocation)
- Dynamic threads (heap allocation)
- Priority levels: 0-255 (higher = higher priority)

When to use: High-performance applications, robotics, automotive.
